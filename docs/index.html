What viewers experience

Voice navigation

Say "go to slide 5", "next", or "back" — the avatar navigates instantly via speech recognition.

Real-time Q&A

Ask questions about any slide. The avatar answers from structured domain data — never hallucinated.

Avatar sees your slides

VLM screen capture lets the avatar describe charts, colors, and layout it actually sees on screen.

Auto-advance

Slides advance automatically after the avatar finishes speaking. Pauses when you engage.

Session memory

Returning viewers pick up where they left off. The avatar remembers their interests and progress.

Single HTML file

No server, no build tools, no runtime dependencies. Share via a stable Kaltura short link.

Get started in 30 seconds

Install the skill, point it at a PDF, and let it build everything.

# Install the skill
/install github.com/kaltura/avatar-presentation-skill

# Generate a presentation
/avatar-deck ./deck.pdf https://your-avatar-url
