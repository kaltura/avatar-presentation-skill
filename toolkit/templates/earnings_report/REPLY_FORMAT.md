# Reply Format — Earnings Report

## TTS Pronunciation Table

Use these pronunciations when speaking. The caption system will display the correct text.

| Say This (Phonetic) | Displays As |
|---------------------|-------------|
| gap | GAAP |
| none gap | Non-GAAP |
| eebeetdaa | EBITDA |
| year over year | YoY |
| quarter over quarter | QoQ |
| A R R | ARR |
| R P O | RPO |
| N D R | NDR |
| E P S | EPS |
| Net Dollar Retention | NDR |
| compound annual growth rate | CAGR |

## Silence Handling

When the user is silent after your turn:
- Do NOT ask "Are you still there?" or "Can you hear me?"
- Instead, offer a natural follow-up related to the current slide
- Examples:
  - "Would you like me to break down any of these numbers further?"
  - "I can also compare this to prior quarters if that's helpful."
  - "Shall I move to the next section, or would you like to explore this data more?"

## Navigation Command Format

When navigating, use ONLY these exact phrases (including the period):
- "Navigating to slide [N]."
- "Moving to the next slide."
- "Going back to the previous slide."
- "Let me show you slide [N]."
- "Ending presentation now."
