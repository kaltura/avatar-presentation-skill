# Summary Prompt — Training

Generate a structured post-session summary for the Training team. Under 400 words. Dense and actionable.

## Required Sections

### Session Metadata
- Date and time
- Duration
- Modules/slides covered (list by number)
- Device type

### Learner Profile
- Inferred experience level (beginner, intermediate, advanced)
- Prior knowledge demonstrated
- Learning style signals (visual, conceptual, example-driven)

### Modules Completed
- List each section covered with depth indicator
- Note which topics required extra explanation
- Note which topics were grasped quickly

### Questions Asked
- Every question verbatim
- Your answer (summarized)
- Flag questions indicating knowledge gaps (mark as "NEEDS REINFORCEMENT")

### Knowledge Gaps Identified
- Concepts that required repeated explanation
- Misconceptions expressed
- Areas where the learner seemed uncertain

### Comprehension Signals
- Topics where the learner asked follow-up questions (indicates engagement)
- Topics where the learner was silent (may indicate confusion or disinterest)
- Any self-assessment statements ("I think I understand", "This is confusing")

### Recommended Next Steps
- Modules to revisit
- Suggested practice exercises
- Prerequisite gaps to address before advancing

## Constraints
- Under 400 words
- Focus on learning outcomes, not just activity
- Note time spent per section for pacing insights
- Flag any certification-readiness signals
